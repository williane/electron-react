
# Electron with React

This is a template to develop a desktop application using Electron Framework with Reac js.


## Installation

Do the git clone

```bash
  git clone https://github.com/williane/electron-react.git
  cd eletron-react
```
Install all npm dependencies  

```bash
  npm install
```

Start you project using

```bash
  npm start
```
